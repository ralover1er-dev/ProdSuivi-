# ProdSuivi

Application mobile de suivi de production en React Native avec Expo SDK 51 + TypeScript.

## Fonctionnalités

- ✅ Authentification locale (inscription/connexion)
- ✅ Gestion de conteneurs
- ✅ Suivi de produits
- ✅ Stockage local avec SQLite
- ✅ Hashage SHA256 des mots de passe

## Installation

```bash
npm install
# ou
yarn install
```

## Démarrage

```bash
npm start
# iOS
npm run ios
# Android
npm run android
```

## Structure du projet

```
src/
├── types/          # Types TypeScript
├── services/       # Services métier
├── context/        # Context API
├── screens/        # Écrans
└── components/     # Composants
```
